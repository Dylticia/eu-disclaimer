Utilisation du plugin : 
Pour plus de facilité, nous avons automatisé le plugin...
Mais si vous souhaitez le maitriser, vous pouvez indiquer 
ce code php sous la balise body html de la page home principale de votre thème WP :
echo do_shortcode('[eu-disclaimer]');
En décommentant également la fonction associée dans le fichier eu-disclaimer.php

Afin d'utiliser ce plugin vous dézippez le dossier  eu-disclaimer,
vous trouvez dans votre arborescence où se trouve votre wordpress
et vous le placez, dans le dossier plugins, ainsi: 
wordpress\wp-content\plugins
